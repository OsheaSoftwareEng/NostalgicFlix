Nostalgia Flicks

This is a Api that allows users to sign up, update their personal information and make a list of their favorite nostalgic movies. This web application will provide users with an array of nostalgic movies, with the year of creation and genre.

How To Use This API

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Returns a list of all movies /movies GET N/A N/A A JSON object of all the movies
Returns a specific movie by title /movies/:title GET :title N/A A JSON object of one movie
Allows users to search for a movie by genre /users/genres/:genreName GET :genreName N/A returns all movies with the genre searched
Allows a user to create a new account /users/ POST N/A A JSON object of a new user ie.{"name": "John Cena", "savedMovies": [] } A JSON object of a new user added ie. {"name" "Joe Smoe", "savedMovies: []}
Allows a user to be deleted /users/:id DELETE :id N/A Response text saying ie. "user "id number" has been deleted
Allows a user to change their username /user/:id PUT :id A JSON object of the new desired name ie. {"name": "Joe Smoe"} Returns a JSON Object with the new name, the ID, and saved movie ie. { "id": "1","name": "Joe Smoe","savedMovies": []}
Returns movies by directors /movies/directors/:directorName GET :directorName N/A A JSON object of all the movies with that searched director
Returns all current users /users GET N/A N/A A JSON object of all the current users
Saves a users favorite movie /users/:id/:movieTitle POST :id/:movieTitle A JSON object of the name and the savedMovies empty array ie. {"name": "Robert Kelly","savedMovies": []} A text message saying the movie has been added to your favorites
Removes a users favorite movie /users/:id/:movieTitle DELETE :id/:movieTitle N/A A text message saying the movie has been removed to your favorites
